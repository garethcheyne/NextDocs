We have introduced a new Quote Won Process to streamline and improve the experience when closing a quote as "Won".

## Key Features

### File Upload Capability

You can now attach supporting documents (such as purchase orders or signed quotes) directly during the quote closure process.

**Benefits:**

- Centralized document storage
- Improved audit trail
- Easy access to supporting documentation
- Streamlined workflow

### Google Address Autocomplete

The delivery address field now supports Google Places Autocomplete, making it faster and easier to enter accurate addresses.

**Features:**

- Real-time address suggestions
- Reduced typing and errors
- Standardized address format
- Improved data quality

### Validated Ship-To Address Selection

You can select from previously validated Ship-To addresses that have been used in past shipments from Business Central, ensuring consistency and reducing errors.

**Advantages:**

- Consistency with historical data
- Reduced address errors
- Faster address entry
- Integration with Business Central data

## Process Improvements

### Streamlined Workflow

The enhanced Quote Won process provides a more efficient and user-friendly experience:

1. **Quote Closure** - Mark quote as "Won"
2. **Document Upload** - Attach supporting files
3. **Address Selection** - Use autocomplete or validated addresses
4. **Completion** - Process completes with improved data quality

### Data Quality Enhancements

These improvements are specifically designed to:

- Improve data quality across the system
- Reduce manual entry errors
- Ensure consistency with Business Central
- Support seamless integration workflows

- **API Integration**: Google Places API

- **Address Validation**: Against BC Ship-To addresses
