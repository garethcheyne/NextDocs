# Content Creator Guide

Welcome to the NextDocs Content Creator Guide! This guide will help you create beautiful, well-structured documentation for your projects.

## What You'll Learn :rocket:

- **Quick Start**: Get up and running in 5 minutes
- **Repository Structure**: How to organize your documentation files
- **Navigation**: Configure menus and document ordering
- **Markdown**: Write rich content with headers, lists, code, and more
- **Icons**: Add visual flair with inline icons
- **Publishing**: Connect your docs to NextDocs

## Who This Guide Is For :target:

This guide is for **content creators** - people who write and maintain documentation. You don't need to know anything about Docker, databases, or web development. Just focus on creating great docs!

## What Is NextDocs? :lightbulb:

NextDocs is a documentation platform that:

- Syncs with your GitHub repository automatically
- Renders Markdown with advanced features (Mermaid diagrams, code highlighting)
- Provides powerful search across all your docs
- Supports multiple icon libraries for visual enhancement
- Handles versioning and categories

## Getting Started :compass:

1. **New to NextDocs?** Start with [Quick Start](./quick-start.md)
2. **Planning your docs?** Check out [Repository Structure](./structure.md)
3. **Need examples?** Download the sample template from this guide
4. **Ready to publish?** See [Publishing Your Docs](./publishing.md)

## Need Help? :help-circle:
