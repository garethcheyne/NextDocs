# FluentUI Icons

Browse Microsoft's FluentUI icon library. All icons use `:#fluentui IconName:` syntax.

## Usage :zap:

Add the `#fluentui` prefix before the icon name:

```markdown
:#fluentui IconName: Your text here
```

**Example**:
```markdown
:#fluentui Add: Create new item
:#fluentui Delete: Remove item
```

**Result**:
:#fluentui Add: Create new item  
:#fluentui Delete: Remove item

## Important Notes :alert-triangle:

- **Case sensitive**: Icon names must match exact casing (e.g., `Add` not `add`)
- **PascalCase**: Most icons use PascalCase (e.g., `CheckMark`, `Settings`)
- **Space required**: Put a space after the icon name before the colon

## Tips :lightbulb:

- **Always check casing**: `Settings` works, `settings` doesn't
- **Use official names**: Visit Microsoft's FluentUI docs for exact names
- **Prefix required**: Don't forget `#fluentui` before the icon name
- **Space after name**: `:#fluentui Add:` not `:#fluentui Add:`

## Finding Icons :search:

1. **Search this page** - Use Ctrl+F (Cmd+F) to find keywords
2. **Check Microsoft docs** - [Official FluentUI Icons](https://developer.microsoft.com/en-us/fluentui#/styles/web/icons)
3. **Try common names** - Most UI elements have intuitive names
4. **Check Office equivalents** - Many Office icons are available
